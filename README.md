# 🌸 Flower Shop Project

A full-stack web application for managing and browsing flowers.

## 🚀 Features
- **User View:** Browse flower collections and add items to cart.
- **Admin Dashboard:** Secure login for admin to manage products (add and delete flowers).
- **Responsive Design:** Smooth and clean user interface built with Bootstrap.

## 🛠️ Tech Stack
- **Frontend:** React.js, Bootstrap, CSS3
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
